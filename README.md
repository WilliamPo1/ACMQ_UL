## Installation

Vous pouvez installer la librairie ACMQ à partir de [Github](https://github.com/vincentarelbundock/ACMQ) en installant la librairie `remotes`::

```r
install.packages('remotes')
```

Ensuite,

```r
library(remotes)
install_github("vincentarelbundock/ACMQ")
```
